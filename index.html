What is React.js



						
    
							
  • React is a JavaScript library that can be used to assist with web development.
    • 
							
    
							
  • React allows for the creation of reusable user interface components.
    • 
							
    
							
  • Initially, React was developed by Facebook for their own use case.
    •




When should you use React.js



						
    
							
  • When you need to display a significant amount of components on a screen.
    • 
							
    
							
  • Examples: Data Visualization, Dashboards, DOM Engagement.
    • 
							
    
							
  • Angular and React share many similarities, consider react for simple single page applications.
    •




Why should you learn React.js?



						
    
							
  • The demand for React developers is growing at a fast pace.
    • 
							
    
							
  • As a developer, you should focus on staying up to date with the newest, popular technologies.
